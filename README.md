# Fokus ⏱️

Fokus é um temporizador de produtividade inspirado na técnica Pomodoro 🍅.

A proposta é simples: transformar suas sessões de estudo ou trabalho em blocos de foco real, com pausas estratégicas para manter energia e consistência durante o dia ⚡.

## O que esse projeto entrega 🚀

- Modos de sessão: foco, descanso curto e descanso longo
- Cronômetro regressivo com atualização em tempo real
- Controle de iniciar, pausar e continuar
- Troca de contexto visual conforme o modo selecionado
- Música ambiente opcional para concentração
- Efeitos sonoros para play, pause e fim de tempo

## Tecnologias utilizadas 🛠️

- HTML5
- CSS3
- JavaScript (vanilla)

## Estrutura do projeto 📂

- index.html: estrutura da interface
- styles.css: layout, tema e responsividade
- script.js: regras do cronômetro, contexto e áudios
- images/: imagens e ícones da interface
- sounds/: música de fundo e efeitos sonoros

## Como executar ▶️

1. Clone ou baixe este repositório.
2. Abra a pasta no VS Code.
3. Execute o arquivo index.html no navegador.

Opcional: use a extensão Live Server para recarregamento automático durante o desenvolvimento.

## Como usar 🎯

1. Escolha o modo: foco, descanso curto ou descanso longo.
2. Clique em Começar para iniciar a contagem.
3. Clique em Pausar para interromper.
4. Clique em Continuar para retomar de onde parou.
5. Ative Música para criar um ambiente sonoro durante a sessão.

## Modos de sessão em detalhes 🧠

### Foco

Modo principal para execução de tarefas importantes.

- Tempo padrão: 25 minutos
- Objetivo: concentração profunda em uma única atividade
- Dica prática: escolha uma tarefa específica antes de iniciar para evitar distrações

### Descanso curto

Pausa rápida para recuperar energia sem perder ritmo.

- Tempo padrão: 5 minutos
- Objetivo: reduzir fadiga mental entre blocos de foco
- Dica prática: levante, beba água e descanse os olhos da tela

### Descanso longo

Pausa maior para recuperação depois de vários ciclos de foco.

- Tempo padrão: 15 minutos
- Objetivo: renovar atenção e diminuir sobrecarga cognitiva
- Dica prática: faça uma pausa longe do ambiente de trabalho para voltar com mais clareza
